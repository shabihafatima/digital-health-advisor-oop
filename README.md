# Digital Health Advisor (OOP)
A console-based Digital Health Advisor developed in **C++** using **Object-Oriented Programming (OOP)**. The application provides user authentication, symptom assessment, BMI calculation, personalized health tips, and file handling through an interactive menu-driven interface.
## Features
- User Registration & Login
- Password Recovery
- Symptom Checker
- BMI Calculator
- Personalized Health Tips
- Diet, Exercise & Mental Health Tips
- File Handling
- Menu-Driven Interface
## Technologies
- C++
- Object-Oriented Programming (OOP)
- File Handling
